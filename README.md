# AgroPathNav
Con el fin de aportar soluciones prácticas e innovadoras para la gestión de invernaderos y la optimización de procesos agrícolas. se propone siguiente proyecto en colaboración con el grupo de investigación GEPAR:

Se propone un vehículo de dos ruedas que se desplaza a lo largo de dos surcos en la tierra, garantizando su trayectoria mediante un sistema de navegación. Para lograrlo, se implementará un sensor LiDAR que permita detectar los límites del surco y corregir la ruta, evitando que el vehículo se desvíe.

El vehículo estará equipado con un conjunto de sensores ambientales que permitirán la medición de variables relevantes dentro del invernadero, tales como temperatura, humedad y luminosidad. De esta forma, el sistema no solo funcionará como un medio de transporte guiado, sino también como una herramienta de monitoreo de las condiciones necesarias para el crecimiento óptimo de los cultivos.
Inicialmente por temas prácticos se realizará un prototipo a escala del carro final (como los de a competencia de analógica) o se preguntará la posibilidad de usar un modelo de gran tamaño que dispone el grupo de investigación (La parte mecánica) 


## Motivación
La motivación surge de una problemática actual de una de las aplicaciones de los prototipos hechos dentro del grupo de investigación, donde ya disponen de un robot que posee el problema de desviarse de su camino debido a la inestabilidad del terreno.
Adicionalmente tenemos gran interés en apoyar el crecimiento, desarrollo e innovación de la industria floricultora o agrícola que tiene nuestro país. Esta industria genera mucho desgaste para quienes se encuentran en las labores de siembra ya que deben mantener posiciones incomodas durante todo el día y puede ser degenerativo a largo plazo.
